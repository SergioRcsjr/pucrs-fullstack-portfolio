# 🚨Estudo-de-Arquitetura-de-Software-em-Nuvem🚨

# Sobre a Disciplina

Introdução à arquitetura de software em nuvem, com foco em datacenters distribuídos, provedores de nuvem, containers, eficiência de performance, backup na nuvem e sustentabilidade em TI.

# Conceitos chave

- Nuvem e datacenters: Nuvem usa estruturas de datacenters distribuídos, sem renunciar a eles  
- Provedor de nuvem: Entidade que mantém serviços disponíveis para partes interessadas  
- Container: Virtualização em nível de OS para implantar e executar apps distribuídos  
- AWS: Maior variedade de tipos de máquinas virtuais para computação  
- Eficiência de performance: Uso eficiente de recursos, adaptando a demandas e evolução tecnológica  
- Backup na nuvem: Essencial, mesmo em ambientes cloud  
- Acesso remoto: Computação em nuvem via internet, com armazenamento em BD importante e foco em sustentabilidade

# Ferramentas iniciais

AWS (máquinas virtuais, serviços), containers para virtualização, ferramentas de backup e monitoramento de performance
