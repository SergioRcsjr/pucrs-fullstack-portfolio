# 🚨Estudo-de-Banco-de-Dados-Relacional🚨

# Sobre a Disciplina

Introdução aos bancos de dados relacionais com foco em modelagem conceitual (ER), modelo lógico, SQL (DDL + DML) e mapeamento objeto-relacional (ORM) com Python.

# Conceitos Principais

- Modelagem: Diagrama Entidade-Relacionamento (MER) → Modelo Lógico → Modelo Físico  
- SQL: DDL (CREATE, ALTER, DROP) e DML (SELECT, INSERT, UPDATE, DELETE)  
- Constraints: PK, FK, UNIQUE, NOT NULL, CHECK  
- Joins (INNER, OUTER, CROSS), GROUP BY, HAVING, subconsultas  
- Funções de agregação (COUNT, SUM, AVG, MAX, MIN)  
- ORM com SQLAlchemy (Python)  
- Index, Sequence, integridade referencial
