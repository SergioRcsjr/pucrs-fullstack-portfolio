# 🚨Estudo-de-Fundamentos de computação e algoritmos🚨

## Sobre a Disciplina

Introdução aos fundamentos de computação, algoritmos e estruturas de dados. Foco em resolver problemas de forma eficiente, analisar complexidade e utilizar corretamente estruturas lineares e hierárquicas.

# Conceitos Principais

- Pensamento computacional e resolução de problemas  
- Complexidade algorítmica (Big-O, melhor/caso médio/pior caso)  
- Estruturas de dados: Array, Pilha (LIFO), Fila (FIFO), Lista Encadeada, Map, Set e Árvore Binária  
- Recursão  
- Busca (linear e binária) e ordenação  
- Performance, latência, perceived performance e trade-offs  
- Práticas: programação no navegador, VS Code, GitHub, debugging e refactoring
