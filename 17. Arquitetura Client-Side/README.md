# 🚨Estudo-de-Arquitetura-Client-Side🚨

# Sobre a Disciplina

Introdução à arquitetura client-side, com foco em camadas front-end (HTML, CSS, JS), back-end e intermediária, convergências como mobilidade, nuvem e IoT, padrões arquiteturais (três camadas, limpa, hexagonal) e elementos para comunicação eficiente entre clientes e servidores.

# Conceitos chave

- Arquitetura client-side: Estrutura com front-end, back-end e intermediária para sistemas distribuídos  
- Business agility: Capacidade de adaptar rapidamente a mudanças de mercado e necessidades de clientes  
- Fundamentos web: URL/URI/URN para identificação de recursos, CDN para entrega de conteúdo, protocolos HTTP para requisições  
- Padrões arquiteturais: Três camadas (comunicação entre camadas), onion (estrutura em camadas), hexagonal (adaptadores de entrada/saída)  
- Elementos essenciais: Dispositivos, plataformas (browsers/OS), mensageria (chatbots), design system para consistência visual  
- Exemplos práticos: Componentes React, roteamento com Next.js, integração de APIs e templates pré-prontos  

# Ferramentas iniciais

HTML/CSS/JavaScript, React, Next.js, CDN, meta tags, design systems
