# 🚨Estudo-de-Projeto-em-Desenvolvimento-de-Aplicação-WEB-Parte-1🚨

# Sobre a Disciplina

Disciplina de projeto integrador do curso Full Stack.  
Foco em desenvolver uma aplicação web real (ou um memorial reflexivo) aplicando todos os conhecimentos adquiridos: front-end, back-end, banco de dados, arquitetura, DevOps e boas práticas.  
Construção de uma solução mínima viável (MVP) para um problema real, com mockup, arquitetura, validação e experimentação das técnicas estudadas.

# Conceitos Principais

- 3 eixos de formação: Fundamentos de Computação → Tecnologias de Desenvolvimento → Tópicos Avançados  
- Desenvolvimento de solução tecnológica (ou memorial)  
- Mockup + arquitetura + estratégia de validação  
- Stacks modernas: MEAN, MERN, LAMP  
- Boas práticas: versionamento, feedback do orientador, reflexão técnica e comportamental

# Exemplos de Tecnologias

React, Node.js, Express.js, MongoDB, PostgreSQL, TypeScript, Angular, GitHub, Heroku, Vercel
