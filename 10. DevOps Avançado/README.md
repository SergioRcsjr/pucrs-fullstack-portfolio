# 🚨Estudo-de-DevOps-Avançado🚨

# Sobre a Disciplina

Aprofundamento em cultura DevOps com foco em Entrega Contínua (CD), contêineres, orquestração, infraestrutura como código (IaC) e observabilidade. Ênfase em automação, integração e entrega contínua, monitoramento e feedback para entregar software de forma rápida, segura e confiável.

# Conceitos Principais

- Cultura DevOps (não é ferramenta, cargo ou só CI/CD)  
- CI/CD (Integração e Entrega/Implantação Contínua)  
- Contêineres (Docker) e orquestração (Kubernetes, ECS, EKS)  
- Infraestrutura como Código (IaC) – Terraform, Ansible  
- Serverless e Fargate  
- Observabilidade (métricas, logs, tracing)  
- Feedback contínuo e troubleshooting  
- Pipelines (GitHub Actions, Jenkins, AWS CodePipeline)

# Ferramentas

GitHub Actions, Docker Compose, Kubernetes, Terraform, Ansible, Jenkins, AWS (ECS, EKS, Fargate, CloudWatch, Elastic Beanstalk)
