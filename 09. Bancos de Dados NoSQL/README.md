# 🚨Estudo-de-Banco-de-Dados-NoSQL🚨

# Sobre a Disciplina

Introdução aos bancos NoSQL com foco em Big Data (volume, velocidade, variedade, validade, volatilidade, valência), Teorema CAP (Consistency, Availability, Partition Tolerance) e os principais modelos: documentos, chave-valor, colunar e grafos. Comparação com bancos relacionais, trade-offs, escalabilidade e casos de uso reais.

# Docentes: 
- Vinícius Kroth (Professor Convidado – especialista em SOA, Microservices, MongoDB, Redis, Elasticsearch e AWS)  
- Eduardo Henrique Pereira de Arruda (Professor PUCRS – fundador da Doc.Space, especialista em modelagem e sistemas)

# Conceitos Principais

- Teorema CAP e modelos ACID × BASE  
- Tipos de NoSQL: Document (MongoDB), Key-Value (Redis), Columnar (Cassandra), Graph (Neo4j)  
- Escalabilidade: sharding, replicação, master-slave, peer-to-peer  
- Modelagem flexível, consistência eventual, trade-offs  
- Casos de uso: redes sociais, time-series, recomendações, detecção de fraude

# Ferramentas

MongoDB, Cassandra, Redis, Neo4j (Cypher), Astah, BRModelo
