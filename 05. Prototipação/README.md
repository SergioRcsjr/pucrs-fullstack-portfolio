# 🚨Estudo-de-Prototipação🚨

## Sobre a Disciplina

Introdução à prototipação rápida para validar ideias e hipóteses antes de desenvolver de fato. Foco em transformar rascunhos em protótipos testáveis, obter feedback precoce e reduzir riscos.

# Conceitos chave

- Prototipação = criar versões preliminares (baixa/alta fidelidade) para testar e aprender rápido  
- Objetivo principal: feedback precoce → aprender com o mínimo esforço  
- MVP= menor coisa viável para validar valor (não é beta)  
  Ciclo Lean Startup: Build → Measure → Learn  
- Validação com clientes 
  Perguntas boas: foco no passado e dores reais  
  Perguntas ruins: hipotéticas / “você usaria?”  
  Dicas: ouça 80%, use Mapa da Empatia, priorize dores intensas  
- Teste de proposta de valor
  Tráfego + Landing Page + Call to Action  
  Métricas: AARRR (Acquisition → Activation → Retention → Revenue → Referral)  
- Estratégias rápidas de MVP 
  Entrevistas, Fake Door, Concierge, Crowdfunding, Landing Page

# Ferramentas iniciais

Papel + caneta, celular (fotos/vídeos), Prott → Figma / Adobe XD
