# 🚨Estudo-de-Design-Thinking-na-Criação-de-Produtos-e-Serviços🚨

# Sobre a Disciplina

Introdução ao Design Thinking como abordagem para criação de produtos e serviços, com foco em colocar o ser humano no centro do processo, etapas iterativas para inovação e resolução de problemas complexos de forma criativa e colaborativa.

# Conceitos chave

- Design Thinking: Abordagem humana-centrada para inovação, focada em empatia, colaboração e experimentação  
- Etapas principais: Empatizar (entender usuários), definir (problema claro), idear (gerar ideias), prototipar (criar soluções tangíveis), testar (validar com feedback)  
- Empatia: Compreender dores, necessidades e motivações dos usuários via entrevistas e observações  
- Definição: Sintetizar insights em problemas bem definidos, usando personas e mapas de empatia  
- Ideação: Brainstorming criativo, priorização de ideias viáveis e inovadoras  
- Prototipação e teste: Criar versões rápidas/low-fidelity para iterar com base em feedback real  
- Benefícios: Redução de riscos, foco em valor para o usuário, integração com lean/agile para produtos/serviços sustentáveis  

# Ferramentas iniciais

Mapas de empatia, personas, journey maps, brainstorming tools, Figma/Adobe XD para protótipos, canvas de proposta de valor
