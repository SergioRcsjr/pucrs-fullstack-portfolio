# 🚨Estudo-de-POO🚨

# Sobre a Disciplina

Disciplina de Programação Orientada a Objetos (POO) com foco em JavaScript (ES6+).  
Ensina a pensar e programar com objetos, classes, herança, polimorfismo, encapsulamento e abstração, além de conceitos avançados como protótipos, funções de alta ordem, async/await, módulos e JSON.

# Conceitos Principais

- Paradigmas: estruturada → procedural → Orientada a Objetos  
- Classes, atributos, métodos, visibilidade, construtores  
- Herança, polimorfismo (sobrescrita), encapsulamento, abstração  
- Protótipos, this, new, classes (ES6)  
- Funções de alta ordem (map, filter, reduce, forEach)  
- Arrow functions, async/await, promises, módulos (import/export)  
- JSON, exceções (try/catch), desestruturação
