# 🚨Estudo-de-Técnicas-Ágeis-de-Programação🚨

# Sobre a Disciplina

Disciplina sobre técnicas ágeis de programação, com foco em fundamentos da agilidade (manifesto, princípios), metodologias ágeis, Extreme Programming (XP), Test Driven Development (TDD) e Behaviour Driven Development (BDD).

# Conceitos chave

- Fundamentos agilidade: Primórdios, manifesto ágil, princípios (aprendizado, colaboração, ciclos de experimentação/falhas)  
- Metodologias ágeis: Panorama, programação em par para troca/colaboracão  
- Extreme Programming (XP): Características, valores, práticas (refatoração para qualidade/design, ciclo de melhoria contínua)  
- Test Driven Development (TDD): Origens, codificar-testar-projetar, benefícios/armadilhas, variações, independência entre testes  
- Behaviour Driven Development (BDD): Origens/princípios, BDD vs TDD, benefícios/armadilhas, foco em valor ao usuário e produção  
- Boas práticas: Identificar perguntas para conectar com problema, saber "pronto" só em produção com valor, vocabulário de refatorações aplicado criteriosamente  

# Ferramentas iniciais

Ferramentas para TDD/BDD (ex: Jest, Cucumber), linguagens como JavaScript/Java para práticas ágeis, refatoração e testes
