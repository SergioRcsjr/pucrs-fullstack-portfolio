# 🚨Estudo-de-Programação-para-Web🚨

# Sobre a Disciplina

Introdução ao desenvolvimento de aplicações web com foco em HTML, CSS e JavaScript. Ênfase em manipulação do Document Object Model (DOM), uso de forms, criação de aplicações responsivas e acessíveis, além de conceitos sobre evolução da web, arquiteturas e protocolos.

# Conceitos chave

- Evolução da web = da 1.0 (estática) à 3.0 (inteligente com IA, segurança e criptografia)  
- Arquiteturas web = cliente-servidor, orientada a serviços (SOA), microserviços (independentes, APIs, flexibilidade)  
- Protocolos = TCP (confiável, conexão) vs UDP (rápido, sem garantia), HTTP/HTTPS, FTP, DNS  
- Dimensões de projeto = estrutural (metas/objetivos), navegacional (navegação), apresentação (visual)  
- Características de apps web = acesso via navegador, dados na nuvem, interatividade, atualizações automáticas, segurança  
- Boas práticas = escalabilidade, flexibilidade, desempenho, padrões e segurança no design  

# Ferramentas iniciais

HTML/CSS/JS, DOM manipulation, forms, responsive design tools (media queries), browsers para testes
